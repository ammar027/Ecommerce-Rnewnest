# RenewNest

**RenewNest** is a modern e-commerce platform dedicated to offering sustainable home goods and eco-friendly products. Our mission is to provide high-quality, stylish items that enhance your home while promoting environmental sustainability.

## Project Overview

RenewNest is an online store offering a curated selection of eco-friendly home products. Our platform is designed with user experience in mind, ensuring a seamless shopping experience from browsing to checkout. 

### Technologies Used
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Hosting:** Vercel

## Features

- **Eco-Friendly Products:** Browse a wide range of sustainable home goods and eco-friendly products.
- **User Accounts:** Register and manage your account, view order history, and track shipments.
- **Shopping Cart:** Easily add products to your cart, update quantities, and proceed to checkout.
- **Search and Filter:** Quickly find products using search and filter options.
- **Responsive Design:** Optimized for various devices and screen sizes.

## Installation

To run RenewNest locally, follow these steps:

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/yourusername/renewnest.git
   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd renewnest
   ```

3. **Install Dependencies:**

   ```bash
   npm install
   ```

4. **Set Up Environment Variables:**
   Create a `.env` file in the root directory and add your environment variables. Example:

   ```env
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret
   ```

5. **Run the Development Server:**

   ```bash
   npm start
   ```

   The application should now be running at `http://localhost:3000`.

## Usage

1. **Visit the Website:**
   Navigate to [RenewNest](https://ecommerce-renewnest.vercel.app/) to explore the available products.

2. **Register/Login:**
   Create an account or log in to manage your orders and preferences.

3. **Browse Products:**
   Use the search bar and filters to find products that meet your needs.

4. **Add to Cart:**
   Add items to your shopping cart and proceed to checkout.

5. **Checkout:**
   Complete your purchase by providing payment details and shipping information.
